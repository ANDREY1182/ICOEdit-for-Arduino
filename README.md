# 🎨 Arduino Icon Editor Pro

Веб-редактор монохромных иконок для Arduino, ESP32 и STM32. Генерирует готовый C++ код с `PROGMEM` прямо в браузере. Работает офлайн, без зависимостей, весит < 15 КБ.

## 🚀 Особенности
- 📐 **Режимы сетки:** 8×8, 16×16 и произвольный размер (до 128×128)
- 🖌 **Рисование:** Клики и перетаскивание, мгновенная подсветка активных пикселей
- 🖼 **Загрузка изображений:** Авто-масштабирование + конвертация в 1-bit монохром (Rec.601 + порог яркости + учёт прозрачности)
- 🔀 **Сдвиг с зацикливанием:** Кнопки `↑ ↓ ← →` перемещают изображение по тороидальной поверхности (пиксели выходят с одного края, появляются с противоположного)
- 🔄 **Инверсия и очистка:** Быстрая корректировка контраста
- 📦 **3 формата экспорта:**
  - `Horizontal MSB` — Adafruit_GFX, TFT_eSPI, U8g2
  - `Horizontal LSB` — XBM-совместимый формат
  - `Vertical MSB` — SSD1306, SH1106, OLED 128×64 (Page Mode)
- 🔃 **Поворот на 90° при экспорте:** Чекбокс для корректного отображения на дисплеях с нестандартной ориентацией GRAM
- 👁 **Предпросмотр 1:1:** Пиксель-в-пиксель с выбором масштаба (1x, 2x, 4x, 8x)
- 💾 **Экспорт:** Копирование в буфер или скачивание `.h` файла
- 🌐 **Офлайн-режим:** Один HTML-файл, не требует сервера, интернета или установки

## 🛠 Быстрый старт
1. Сохраните `index.html` и откройте в браузере (Chrome, Firefox, Edge, Safari).
2. Нарисуйте иконку или загрузите изображение (`🖼 Загрузить`).
3. При необходимости сдвиньте изображение стрелками или инвертируйте цвета.
4. Выберите **Формат экспорта** под вашу библиотеку дисплея.
5. Если на экране иконка повёрнута на 90°, включите `☑ Повернуть 90°`.
6. Нажмите `📋 Копировать` или `💾 Скачать .h` и подключите в проекте:
